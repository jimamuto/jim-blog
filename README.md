## What this blog is

A personal developer blog built from scratch with Next.js. A space to write about coding, homelabbing, and the learning journey.

---

## Features

- Home page with a hero intro, featured post, and category cards
- Writing page that lists blog posts with title, date, and description
- About page
- Navbar with avatar, social links (Twitter, GitHub, Instagram)
- Footer
- MDX support for writing posts in markdown

## Planned

- Dynamic routing so each post has its own URL (`/writing/[slug]`)
- Read posts from markdown files automatically instead of a hardcoded list
- Newsletter signup
- Author section on the home page
